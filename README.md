# PyWebView HTML -> Презентация

Desktop-приложение на `pywebview`:
- вкладка **Сборщик**: принимает URL, собирает HTML-страницы сайта и создает локальную презентацию;
- отображает **progress bar** прогресса загрузки и сборки;
- вкладка **Презентация**: показывает готовую локальную презентацию со встроенными HTML (и их CSS);
- вкладка **Калькулятор**: рабочий калькулятор.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Использование

1. Вставьте стартовую ссылку сайта.
2. Поставьте галочку **Без лимита**, если нужно собрать все доступные страницы; тогда поле лимита блокируется.
3. Если галочка не активна — укажите лимит страниц (по умолчанию 30).
4. Нажмите **Старт**.
5. После завершения откройте вкладку **Презентация**.

Собранные страницы сохраняются в `output/pages`, итоговая презентация — `output/presentation.html`.

## Важно

Если сайт требует авторизацию (например, закрытые страницы Moodle), без входа могут собраться только публично доступные страницы.
